# Air-Quality-Index-ML
Air pollution is one of the major environmental risks to human health. This project predicts the Air Quality Index (AQI) using Machine Learning models based on historical air pollutant data such as PM2.5, PM10, NO₂, CO, SO₂, and O₃.
The goal is to:

Predict AQI values accurately
Classify air quality levels (Good, Moderate, Unhealthy, etc.)
Help authorities and individuals make informed decisions

AQI categories typically include:

AQI      Range	 Category	Health Impact
0–50	   Good   	Minimal impact
51–100	 Moderate	 Acceptable
101–150  Unhealthy for Sensitive Groups	Mild health effects
151–200  Unhealthy	Everyone may experience effects
201–300	 Very       Unhealthy	Health alert
301+	   Hazardous	Serious health risk

🧠 Machine Learning Approach
🔹 Problem Type

Regression (Predict AQI value)

Classification (Predict AQI category)

🔹 Algorithms Used

Linear Regression

Random Forest

Decision Tree

XGBoost

Support Vector Machine (optional)
